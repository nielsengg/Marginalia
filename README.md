## Marginalia
In Marginalia, readers can catalog the books they are currently reading or have already read. While reading, they can create and update notes about the book, with each entry automatically saving the date and time. Once the book is finished, the reader can rate it from 0 to 5 stars and write a final review if desired. In the case of adding a book that was read in the past, the system will save the completion date along with the rating and, optionally, a written analysis. All this information will be stored and displayed on the reader’s profile.

## > Requirements

- [libcurl](https://curl.se/libcurl/)
- Internet connection  
- A `cacert.pem` file for HTTPS support
